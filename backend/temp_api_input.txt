Escribe un algoritmo llamado Seleccion que reciba un arreglo A de tamaño n. El algoritmo debe recorrer el arreglo desde la primera posición hasta la penúltima. En cada iteración, debe buscar el número más pequeño en la parte no ordenada del arreglo (desde la posición actual hasta el final). Una vez encuentre el menor, debe intercambiarlo con la posición actual. Usa la variable min para guardar el índice del menor
